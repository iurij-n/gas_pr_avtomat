2. «Магический» файл авторизации (auth.json)

Чтобы не заставлять бота каждый раз проходить круги ада с Госуслугами, мы один раз сохраним состояние браузера.

Как сделать:
Запустите специальную команду записи с сохранением состояния:
PowerShell

playwright codegen --save-storage=auth.json https://ej.sudrf.ru/

Откроется браузер. Вы руками проходите авторизацию через ЕСИА. Как только увидите личный кабинет ГАС «Правосудие» — закрывайте браузер. У вас в папке появится файл auth.json.

В коде бота теперь будете использовать его так:
Python

context = browser.new_context(storage_state="auth.json")

Бот будет открывать сайт уже залогиненным.


1. Запуск записи с загрузкой авторизации

Выполните команду в терминале:
PowerShell

playwright codegen --load-storage=auth.json https://ej.sudrf.ru/

Браузер откроется, и вы (если сессия жива) сразу увидите интерфейс авторизованного пользователя.



Как сделать правильно:

Выполните в терминале обновленную команду:
PowerShell

playwright codegen --load-storage=auth.json -o my_bot.py https://ej.sudrf.ru/
